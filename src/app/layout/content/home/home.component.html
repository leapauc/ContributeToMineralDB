Bienvenue sur l'application de collaboration à la Base de données minéral et son site !


Cette page permet à toute personne d'aider à l'amélioration de la base de données et du site web sur les minéraux. Le savoir est quelque chose qui devrait être partagé au plus grand nombre. C'est dans cette volonté que l'idée de création d'une base de données minéral à vu le jour


Comment collaborer ?


Vous pouvez à votre échelle collaborée de diverses manières à ce projet.

Nous proposons à travers cette interface que vous puissiez :

  • Ajouter des informations aux sites en :
    • Proposant de nouveaux minéraux ;
    • Proposant de nouvelles thématques pour une page ;
  • Modifier des information du site :
    • Proposant de modifier les informations des minéraux ;
    • Proposant de modifier le contenu d'une page ;
  • Soummettre des suggestions pour améliorer :
    • la base de données ;
    • le site web.
